# Акустические инструменты (Чаши, Гонги, Камертоны)

Инструменты физического резонанса для звуковой терапии и тонкой настройки.

## Камертоны

Самый точный инструмент для индивидуальной настройки (Cosmic Octave / Hans Cousto). Утяжелённые ставят на тело.

| Hz | Название | Описание | Контекст |
| :--- | :--- | :--- | :--- |
| **126.22** | Камертон Солнца | Воля, центр, лидерство, действие. | Солнце / камертон ♪ |
| **136.10** | Камертон OM / Earth Year | Самый распространённый камертон. OM, медитация. | Сердечная / Земля год / камертон ♪ |
| **194.18** | Камертон Earth Day | Утяжелённый — для тела, костей, заземления. | Корневая / Земля день / камертон (тело) ♪ |
| **221.23** | Камертон Венеры | Тонкое восприятие, красота, гармония. | Венера / камертон ♪ |
| **147.85** | Камертон Сатурна | Структура, границы, дисциплина. | Сатурн / камертон ♪ |
| **141.27** | Камертон Меркурия | Коммуникация, ясность речи. | Горловая / Меркурий / камертон ♪ |
| **256.00** | Master Fork — Steiner C | Историческая нота C=256 Hz. Базовый научный строй. | nota C / камертон 256 ♪ |
| **128.00** | Otto 128 (тело) | Низкий камертон для глубокой работы с телом, фасциями. | камертон (тело) ♪ |

| **167.00** | Тибетская чаша E | Реальная антикварная чаша E (≈ 167 Hz, +28 cents). | Солн. сплетение / nota E / тибетская чаша ♪ |

| **392.00** | Хрустальная чаша G | Самовыражение, правда. | Горловая / nota G / хрустальная чаша ♪ |
