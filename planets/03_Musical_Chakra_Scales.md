# Нотные системы чакр (440Hz vs 432Hz)

Сравнение стандартного строя и альтернативного "природного" строя.

## Стандарт 440 Hz

| Hz | Нота | Чакра | Описание |
| :--- | :--- | :--- | :--- |
| **261.63** | C4 | Корневая | Теплота, стабильность, заземление. |
| **293.66** | D4 | Сакральная | Поток, чувственность. |
| **164.81** | E3 | Солн. сплетение | Воля, уверенность, центр силы. |
| **349.23** | F4 | Сердечная | Любовь, интеграция. |
| **392.00** | G4 | Горловая | Самовыражение, правда. |
| **440.00** | A4 | Третий глаз | Международный стандарт настройки. |
| **493.88** | B4 | Коронная | Единство. |

## Альтернатива 432 Hz

| Hz | Нота | Чакра | Описание |
| :--- | :--- | :--- | :--- |
| **256.00** | C3 | Корневая | Заземление, безопасность. (2^8). |
| **288.00** | D3 | Сакральная | Творчество, сексуальность. |
| **167.00** | E3+ | Солн. сплетение | Практическая частота тибетских чаш. |
| **171.00** | F3 | Сердечная | Любовь, сострадание. |
| **192.00** | G3 | Горловая | Самовыражение, коммуникация. |
| **216.00** | A3 | Третий глаз | Интуиция, ясность. |
| **432.00** | A4 | Третий глаз | Эталон альтернативной настройки. |
| **240.00** | B3 | Коронная | Единство, духовное пробуждение. |
