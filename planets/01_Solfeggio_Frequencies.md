# Частоты Сольфеджио (Solfeggio)

Древняя система 9 тонов, популяризована д-ром Джозефом Пулео. Оптимальный инструмент: цифровой генератор / аудиотрек.

| Hz | Название | Описание | Контекст |
| :--- | :--- | :--- | :--- |
| **174** | Освобождение от боли | Снижение боли и стресса, ощущение безопасности, анестезирующий эффект. | Корневая / синтезатор |
| **285** | Восстановление тканей | Регенерация клеток и тканей, связь с физическим телом. | Сакральная / синтезатор |
| **396** | Освобождение от страха | Освобождение от страха и вины, трансформация горя, заземление. | Корневая / синтезатор |
| **417** | Трансформация | Фасилитация перемен, разрушение негативных паттернов, очищение. | Сакральная / синтезатор |
| **528** | Исцеление / ДНК | «Частота чуда», любовь, ясность, репарация ДНК по версии исследователей. | Солнечное сплетение / синтезатор |
| **639** | Связь и отношения | Гармонизация отношений, коммуникация, баланс эмоций. | Сердечная / синтезатор |
| **741** | Очищение и выражение | Детоксикация, пробуждение интуиции, решение проблем. | Горловая / синтезатор |
| **852** | Духовный порядок | Возврат к духовному порядку, пробуждение интуиции, связь с высшим Я. | Третий глаз / синтезатор |
| **963** | Единство / Коронная | Активация шишковидной железы, единство с Вселенной, чистое сознание. | Коронная / синтезатор |
