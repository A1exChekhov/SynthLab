# Channel Splitter · Errarium™

Аудио-сплиттер для macOS: захватывает звук системы / конкретного приложения / интернет-радио,
обрабатывает его (эквалайзер, эффекты, авто-калибровка задержек) и **раздаёт на несколько колонок
одновременно** — каждой свою роль (левый / правый / моно / стерео), громкость и задержку.

Нативный порт Windows-версии на **Swift + SwiftUI + AVAudioEngine + Core Audio**. Веб-интерфейс
(WKWebView) повторяет оригинальный UI 1:1. Собирается **без Xcode** — только Command Line Tools.

> macOS 14.2+ (Sonoma и новее) · Apple Silicon / Intel

---

## Установка

1. Скачай `ChannelSplitter.dmg` из раздела [**Releases**](../../releases).
2. Открой DMG → перетащи **Channel Splitter** в **Applications**.
3. Первый запуск: правый клик по приложению → **Open** (приложение подписано ad-hoc, поэтому
   Gatekeeper спрашивает подтверждение один раз).
4. Выдай разрешения по запросу:
   - **Микрофон** — для авто-калибровки задержек колонок;
   - **Запись экрана / системного звука** — для захвата звука системы и приложений
     (Core Audio process tap, обязательно для источника «System Audio» и per-app);
   - **Автоматизация → Music** (опц.) — чтобы при переключении источника ставить Apple Music на паузу.

## Возможности

- **Раздача на несколько колонок** — по одному `AVAudioEngine` на устройство; роль L / R / Mono / L-R,
  громкость, mute, мастер, инверсия фазы, сабвуфер с кроссовером. Толерантный запуск: сбойная колонка
  не роняет остальные.
- **Источники:** весь системный звук, **конкретное приложение** (per-app loopback), физический вход,
  **интернет-радио** (Tuner). Приоритет последнего запущенного приложения (авто-переключение).
- **Интернет-радио через сплиттер** — собственный потоковый декодер (ICY → AudioFileStream →
  AudioConverter → PCM), радио проходит через EQ/эффекты/спектр; реальный кодек и битрейт потока.
- **Эквалайзер** — 12 полос (20 Гц…20 кГц), пресеты, спектр (vDSP FFT).
- **Эффекты:** Bass Boost, Spatial (M/S), 3D (Haas), 7.1 Surround, Mono-Bass, Reverb, и др.
- **Авто-калибровка задержек** — лог-свип в каждую колонку, запись микрофоном, кросс-корреляция.
- **Плеер** — now-playing (название, исполнитель, обложка, время), реальный формат потока,
  транспорт (play / pause / stop с перемоткой в начало / prev / next).
- **Анализатор / цветомузыка** — столбики спектра (цвет / ч-б) или WebGL-цветомузыка прямо в окне
  (кнопка GPU, переключение эффектов), плюс полноэкранный режим «Open GPU».
- **Мини-плеер** — компактное окно поверх остальных: вызывается кнопкой **Mini** в нижнем ряду
  или из трея; кнопка ▣ показывает/скрывает главное окно; полностью отражает now-playing,
  включая радио (станция, трек, обложка песни, время, битрейт).
- **Трей**, тёмная/светлая темы, 1/2 колонки. **Сохранение между запусками:** свёрнутые блоки,
  последняя станция тюнера, входы/выходы/микрофон, тема и раскладка.

## Сборка из исходников

```bash
cd mac/ChannelSplitterMac
./build.sh release        # swift build → ChannelSplitter.app + ad-hoc подпись
open ChannelSplitter.app

./make_dmg.sh release     # собрать установщик ChannelSplitter.dmg
```

Подробности реализации (фазы, параметры DSP) — в [`mac/ChannelSplitterMac/README_macOS.md`](mac/ChannelSplitterMac/README_macOS.md).
