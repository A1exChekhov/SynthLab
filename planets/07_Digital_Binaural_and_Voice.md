# Цифровой синтез, Бинауральные биения и Голос

Генерация точных частот (синтезаторы), стимуляция мозга (бинауральные ритмы) и внутренняя вибрация (мантры).

## Бинауральные / монауральные биения

Истинный binaural требует наушников: мозг создаёт разностный ритм из двух разных частот.

| Hz | Название | Описание | Контекст |
| :--- | :--- | :--- | :--- |
| **200.00** | Бинауральный тета 7.8 Hz | Стерео-пара даёт ~7.8 Hz при базе 200 Hz. Медитация, гипноз. ОБЯЗАТЕЛЬНО в наушниках. | бинауральный (наушники) ♪ |
| **136.10** | Бинауральный OM | Стерео-пара в районе OM. Релаксация. | бинауральный (наушники) ♪ |
| **100.00** | Бинауральный альфа ≈8-12 Hz | При базе 100 Hz даёт ~8 Hz (альфа-расслабление). | бинауральный (наушники) ♪ |
| **250.56** | Шуман — октавный тон | Слышимый аналог 7.83 Hz (× 2⁵). Работает без наушников. | генератор |
| **7.83** | Шуман прямой 7.83 Hz | Слышимого тона нет — только бинауральная пара через наушники. | бинауральный (наушники) ♪ |
