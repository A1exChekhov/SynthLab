# Дрон-инструменты и Ритм (Барабаны)

Инструменты длительного удержания поля (монохорды) и ритмические генераторы состояний (барабаны).

## Монохорд / тамбура / шрути-бокс

Длительное непрерывное звучание. Удержание пространства, OM-практики, дыхание.

| Hz | Название | Описание | Контекст |
| :--- | :--- | :--- | :--- |
| **136.10** | Монохорд OM | Длинный смычковый дрон на OM, дыхательные практики. | Сердечная / монохорд ♪ |
| **432.00** | Монохорд A=432 | Дрон в строе 432 Hz. Богатый обертоновый стек. | nota A / монохорд ♪ |
| **261.63** | Монохорд C — корень | Низкий заземляющий дрон. | Корневая / nota C / монохорд ♪ |
| **136.10** | Шрути-бокс OM | Индийский гармониум-дрон под мантру или дыхание. | Сердечная / шрути-бокс ♪ |
| **167.00** | Шрути-бокс E | Шрути для солнечного сплетения через ноту E. | Солн. сплетение / nota E / шрути-бокс ♪ |
| **220.00** | Тамбура A — третий глаз | Индийский струнный дрон. | Третий глаз / nota A / тамбура ♪ |

## Барабаны

Hz здесь — это не несущий тон, а ритмическая пульсация (удары в секунду).

| Hz | Название | Описание | Контекст |
| :--- | :--- | :--- | :--- |
| **80.00** | Фрейм-барабан | Низкий короткий удар, тёмное тело. | фрейм-барабан ♪ |
| **100.00** | Шаманский барабан (Тета 5 Hz) | 5 ударов в секунду. Шаманские путешествия, трансовые практики. | шаманский барабан ♪ |
| **70.00** | Шаманский медленный | Тот же ритм 5 Hz, более низкий несущий тон — глубже. | шаманский барабан ♪ |
| **90.00** | Барабан — мягкий транс (4 Hz) | Мягкое трансовое состояние. | шаманский барабан ♪ |
| **110.00** | Барабан — активный (7 Hz) | Более активная настройка. | шаманский барабан ♪ |
