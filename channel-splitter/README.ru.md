# Channel Splitter — Errarium™

**Channel Splitter** — премиальный аудиомаршрутизатор для Windows. Он захватывает
стереоисточник на ПК и отправляет **левый и правый каналы на независимые устройства**
(например, две Bluetooth‑колонки), сохраняя при этом стереопару (наушники) целиком.
Каждый выход — отдельный WASAPI‑поток со своим буфером, поэтому независимые часы
беспроводных устройств не дают щелчков, а индивидуальная задержка позволяет их свести.

Поставляется одним установщиком с интерфейсом в стиле профессиональной AV‑аппаратуры
(брашированный металл, семисегментные индикаторы, трёхцветные сегментные шкалы),
12‑полосным эквалайзером, стойкой спецэффектов, GPU‑цветомузыкой, плеером «сейчас
играет», сворачиванием в трей и плавающим мини‑плеером. Язык интерфейса
(русский / английский) выбирается при установке.

## Возможности

- **Матрица каналов** — сколько угодно выходов; каждый — **L / R / Mono / L‑R**,
  со своей громкостью, mute, режимом сабвуфера (ФНЧ) и инверсией фазы (Ø).
- **Несколько источников** — системный звук (WASAPI loopback — «что играет ПК») и/или
  физические входы (микрофон, линейный, VB‑Audio CABLE); у каждого громкость, баланс,
  mute и фаза.
- **Компенсация задержки**
  - **Ручной ползунок синхронизации** — тяни, пока колонки не совпадут.
  - **Авто‑калибровка** — микрофон проигрывает тихий свип на каждый выход и
    автоматически выравнивает задержки.
  - **HOLD** — зафиксируй удобную синхронизацию, и программа удерживает её онлайн,
    отслеживая дрейф часов Bluetooth по микрофону (без слышимых сигналов).
- **12‑полосный эквалайзер** с сохраняемыми пресетами (переживают обновления).
- **Стойка спецэффектов** — 2D‑площадки Space (ширина/глубина), Position (панорама/
  дистанция), Tone (тепло‑ярко/драйв), плюс компрессор, моно‑бас и реверберация
  (размер + микс).
- **GPU‑цветомузыка** — полноэкранные / 4K поля в стиле MilkDrop, несколько пресетов и
  цветовых режимов; повторное нажатие кнопки Visualizer переключает пресеты.
- **Плеер «сейчас играет»** — название, исполнитель и реальная позиция из Windows Media
  Session, с управлением (плей/пауза, вперёд, назад, стоп) активным медиаплеером.
- **Системный трей** — закрытие окна сворачивает в трей; меню трея: показать /
  мини‑плеер / транспорт / выход.
- **Плавающий мини‑плеер** — компактная полоска поверх всех окон для управления
  воспроизведением, когда главное окно скрыто.
- **Защита от громкого старта** — первый запуск на безопасной громкости с плавным
  нарастанием; все настройки (громкости, EQ, эффекты, тема, раскладка, язык)
  запоминаются и восстанавливаются.
- **Темы и раскладка** — лицевая панель Dark или Silver (в стиле JVC); стойка в одну
  или две колонки.
- **Двуязычный интерфейс** — русский / английский, включая тултипы. Выбор при установке.

## Установка

1. Запусти **`ChannelSplitter-Setup-2.2.exe`**.
2. Выбери язык интерфейса (русский / английский) в мастере установки.
3. Запусти Channel Splitter.

Настройки и пресеты EQ хранятся в `%APPDATA%\Errarium\ChannelSplitter\` и сохраняются
при обновлениях и переустановках.

### Рекомендуется: завести звук Windows в программу
Чтобы разделять **всё**, что играет ПК, добавь источник **System Audio** (WASAPI
loopback) — без дополнительной настройки. Либо установи **VB‑Audio Virtual Cable**,
сделай вывод Windows по умолчанию **CABLE Input** и добавь **CABLE Output** источником.

## Быстрый старт

1. **Добавь выходы** (Output Matrix → **+ Output**): выбери устройство и роль
   (например, одна колонка **L**, другая **R**; наушники **L‑R** для полного стерео).
2. **Добавь источник** (Pre‑Amp → **+ System** для системного звука или **+ Source**
   для физического входа).
3. Нажми **power (ON)** на нижней панели.
4. **Сведи колонки**: тяни ползунок **SYNC**, либо запусти **Calibrate** с микрофоном,
   либо нажми **HOLD**, чтобы удерживать синхронизацию онлайн.

Все регуляторы работают на лету во время воспроизведения.

## Про Bluetooth

Две **независимые** Bluetooth‑колонки нельзя засинхронизировать сэмпл‑в‑сэмпл навсегда —
у каждой свои часы и своя «гуляющая» задержка. Ручной ползунок и авто‑калибровка задают
сдвиг; **HOLD** постоянно компенсирует дрейф по микрофону. Небольшой остаточный сдвиг —
это физика Bluetooth, а не баг. Для идеальной синхронности — провод или один передатчик
на стереопару.
